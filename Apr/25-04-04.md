- ### **“순환”이란?**
    - 순환은 **은닉 상태(hidden state)**가 반복적으로 전달되면서

        시간의 흐름에 따라 정보를 누적해서 반영하는 구조를 말합니다.

    - 한 시점에서의 은닉 상태는 이전 시점의 은닉 상태를 **함수처럼 받아서 계산**됩니다.

        이게 바로 “순환”이라는 말의 의미예요.

| **질문**            | **설명**                                                           |
| ----------------- | ---------------------------------------------------------------- |
| 은닉 파라미터가 순환하나요?   | 네, RNN은 하나의 은닉 파라미터(Wₓ, Wₕ 등)를 공유하며, 시간의 흐름에 따라 내부에서 순환하며 학습합니다. |
| 은닉층이 여러 개인가요?     | 일반 RNN은 **하나의 은닉층**을 시간축으로 반복합니다. 딥 RNN이면 은닉층이 여러 개 쌓일 수 있어요.    |
| 시간 순환은 어떻게 작동하나요? | 이전 시점의 h_{t-1}을 받아서 현재 h_t를 계산하고, 다음 시점에 넘겨주는 방식입니다.             |

- 아 그리고 시그모이드 함수 형태가 로짓(logit)값에 자연로그를 취해서 만든 함수였음!!!
- ### **로짓(logit)과 시그모이드의 관계**

    > ## 로짓(logit)은 출력층의 “최종 계산 값(=score)”

    - **시그모이드 = 로짓의 역함수**
    - **로짓(logit)** 함수는 다음과 같이 정의됩니다:

            \text{logit}(p) = \log\left(\frac{p}{1 - p}\right)

    - 즉, **확률 p에 자연로그를 취한 odds(승산)** 값입니다.
    - 반대로, 로짓 x 값을 시그모이드에 넣으면 **확률로 변환**됩니다.
